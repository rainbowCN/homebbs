什么是Markdown？

一种轻量级的标记语言，它规定了一些文本的书写格式，就像这样(demo.MD)：

Markdown入门
===========
定义
--------
' '内容
语法
--------
' '内容
工具
--------
' '内容

demo.MD经过转换后生成的demo.HTML

<h1>Markdown入门</h1>
<h2>定义</h2>
<p>内容</p>
<h2>语法</h2>
<p>内容</p>
<h2>工具</h2>
<p>内容</p>


标题标记

===

说明:

等同于#{内容}#语法标记

.MD:

标题
===

HTML:

<h1>标题</h1>

------

说明:

等同于##{内容}##语法标记

.MD:

标题
------

HTML:

<h2>标题</h2>

#

说明:

标题（h1~h6）格式为使用相应个数的“#”作前缀，比如以下代码表示 h3。

.MD:

###标题###

HTML:

<h3>标题</h3>

内容标记

_{内容}_

说明:

被强调的内容

.MD:

_我是被强调的内容_

HTML:

<strong>我是被强调的内容</strong>
